# FlacPlayer Feedback

This is the public feedback tracker for **FlacPlayer**, a native lossless music
player focused on local FLAC playback. Development is currently iPhone first,
with multiplatform support planned.

The source code is maintained privately. This repository exists for user
feedback, support triage, and visible tracking of reported problems and ideas.

## Report Something

Use the matching issue form:

- [Report a bug](https://github.com/AndresASJ/FlacPlayer-Feedback/issues/new?template=bug_report.yml)
- [Request a feature](https://github.com/AndresASJ/FlacPlayer-Feedback/issues/new?template=feature_request.yml)
- [Report playback or audio quality trouble](https://github.com/AndresASJ/FlacPlayer-Feedback/issues/new?template=playback_issue.yml)
- [Report a library or metadata problem](https://github.com/AndresASJ/FlacPlayer-Feedback/issues/new?template=library_metadata_issue.yml)

Before reporting, please check
[existing issues](https://github.com/AndresASJ/FlacPlayer-Feedback/issues) to
avoid duplicates.

## Public Information Warning

Issues and attachments in this repository are public. Do not include:

- Personal file paths, cloud-drive links, or account information
- Music files or copyrighted artwork
- Tokens, passwords, API keys, or diagnostic data containing private details
- Security vulnerabilities or privacy-sensitive reports

For a security or privacy-sensitive issue, do not open a public issue. A
private reporting channel will be published before public distribution of the
app.

## Helpful Reports

Good reports include the FlacPlayer version, device and OS version, clear
reproduction steps, what happened, and what you expected. Audio reports should
also mention the output device or DAC and whether EQ was enabled.
